# 개인정보 처리방침

최종 업데이트: 2025년 1월 31일

## 소개

Sugumemo(이하 「앱」)는 이용자의 프라이버시를 존중하며 개인정보를 보호하기 위해 노력합니다. 본 처리방침은 앱이 수집하는 정보와 이용 방법을 설명합니다.

## 1. 수집하는 정보

### 1.1 앱에 저장되는 정보

앱은 메모·리마인더 내용, 생성 일시, 알림 예정 시각, 리마인더 상태(예정/완료/만료), 설정(마지막 사전 설정 시간 등)을 기기 내 로컬 스토리지(AsyncStorage)에 저장하며 외부 서버로 전송하지 않습니다.

### 1.2 권한 사용

앱은 마이크(음성 입력, 기기 내 처리), 음성 인식(기기에서 음성→텍스트 변환), 알림(리마인더 전송) 권한을 사용합니다.

### 1.3 자동 수집 정보

앱은 기기 정보(유형, OS 버전, 언어) 및 Google AdMob용 광고 식별자를 수집할 수 있습니다.

## 2. 정보 이용

수집 정보는 메모·리마인더, 알림, 음성 입력 및 기술적 처리에만 사용됩니다. 앱은 Google AdMob으로 광고를 표시하며, AdMob은 광고 식별자, 기기 정보, 노출/클릭을 수집·이용할 수 있습니다. 유료 플랜(Pro) 구매·복원에는 RevenueCat을 사용하며, 거래 데이터는 App Store/Google Play를 통해 처리됩니다. [Google 개인정보처리방침](https://policies.google.com/privacy), [RevenueCat 개인정보처리방침](https://www.revenuecat.com/privacy) 참조.

## 3. 정보 공유

앱은 광고(AdMob), 결제 처리(RevenueCat), 법적 요구 시를 제외하고 개인정보를 제3자와 공유하지 않습니다.

## 4. 보관 및 삭제

메모·리마인더 데이터는 이용자 기기에만 저장됩니다. 리마인더는 알림 예정일로부터 30일 후 자동 삭제되며, 50건 초과 시 오래된 데이터부터 삭제됩니다. 앱 삭제 시 모든 데이터가 삭제됩니다.

## 5. 보안

데이터는 기기 내에만 저장되며, 메모 내용은 외부 서버로 전송되지 않고, 해당 시 암호화된 스토리지를 사용합니다.

## 6. 아동

앱은 13세 미만을 대상으로 하지 않으며, 13세 미만 아동의 개인정보를 고의로 수집하지 않습니다.

## 7. 변경

본 방침은 사전 통지 없이 변경될 수 있습니다. 중요한 변경 시 앱 또는 본 페이지에서 안내합니다.
